# 🦀 Oxidust 🌬️
**The "Ghost-Buster" for your Filesystem.**

Oxidust is a cross-platform utility that helps you visualize, manage, and clean disk space across **Linux**, **macOS**, and **Windows**. Built with **Rust** for raw speed and **Tauri** for a lightweight footprint.

### 🚀 Why Oxidust?
* **WinDirStat-style TreeMap:** Instantly spot giant files and "Ghost" processes (deleted files still held by OS).
* **Layered Rule Engine:** OS-specific presets (Ubuntu, Arch, Win, Mac) that "know" exactly where the junk hides.
* **Remote Web UI:** Manage and clean multiple servers or PCs from a single dashboard.
* **Deep Server Clean:** Specialized rules for Docker, Nginx logs, and Inode-heavy directories.
* **Privacy First:** No data leaves your machine unless you use the optional Cloud Sync for presets.
